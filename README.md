Pour lancer le programme il faut prendre les dernieres fichiers disponibles sur le git.
IL faut aussi java 8 ou plus pour lancer le programme.
Un fois les conditions reunie , il faut se placer dans le dossier du projet et lancer la commande suivante dans le terminal :
javac *.java
java Main
Ainsi les programmes se lance et profitez d'une partie de jeu !
